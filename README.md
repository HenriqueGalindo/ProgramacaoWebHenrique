# **OndeTem**

  O OndeTem é um site onde uma série de estabelecimentos de prestação de serviços seriam divididos em tags, como Restaurante, Academia, Loja de roupas, Lavanderia, Shopping, etc, e por tags secundárias (no caso de restaurantes, o tipo de comida que servem, se possuem opções veganas ou para alergias) as quais os usuários podem selecionar e receber as informações de estabelecimentos que possuam essas tags na cidade onde moram.
